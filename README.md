# Tasks

Projetos simples com `Flask`.

## 🌐 Como executar

1. **Clone o repositório**

    ```git
    git clone https://github.com/Davi-1903/Tasks.git
    ```

2. **Instale todas as dependências**

    ```sh
    pip install -r requirements.txt
    ```

3. **Execute a aplicação**

    ```sh
    # Linux/MacOS
    python3 src/app.py

    # Windows
    python src/app.py
    ```
